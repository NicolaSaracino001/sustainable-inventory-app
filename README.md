# S.I.M. - Sustainable Inventory Management 🍎

Benvenuto in S.I.M., un'applicazione Flask per la gestione intelligente del magazzino alimentare.

## 📌 Stato del Progetto
- **Fase attuale:** 1 (Inizializzazione)
- **Obiettivo:** Creare la struttura base e collegare GitHub Desktop.

## 📂 Struttura Cartelle
- `src/models`: Definizioni del database.
- `src/routes`: Logica di navigazione.
- `src/static`: File CSS e immagini.
- `src/templates`: File HTML.
